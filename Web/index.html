RAmazonS3

Last Release: 0.1-4 (15 Feb 2011)

This package provides the basic infrastructure within R for communicating with the S3 Amazon storage server. This is commercial server that allows one to store content and retrieve it from any machine connected to the Internet. The "files" or objects that one creates can be made available to other S3 users or available generally via a URL. To store, access and control access to files on the S3 server, one typically needs to sign up for a login and AWS secret which are needed in most calls. Content that is made accessible to all does not require a login-secret pair.

The package provides functions for most of the common operations available in the S3 interface. This includes

  • finding all buckets belonging to a particular user/login,
  • listing the contents of any bucket,
  • retrieving a file,
  • adding a file
  • removing a file or bucket,
  • copying a file,
  • renaming a file,
  • setting access to a file or bucket,
  • querying meta-information about a file,
